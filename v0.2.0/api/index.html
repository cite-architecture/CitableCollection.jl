CitableCollection.catalogdf — Function

Parse CEX source data into a catalog of CitableCollections.

catalogdf(cexsrc)
catalogdf(cexsrc, delim)

cexsrc must have at least one citecollections block and one citeproperties block.

source
CitableCollection.collectiondfs — Function

Use catalog to parse CEX source data into a Vector of DataFrames.

collectiondfs(cexsrc, catalogdf)
collectiondfs(cexsrc, catalogdf, delim; limitto)

CitableCollection.catalogdata — Function

From a list of CEX blocks, extract all data lines from one or more citecollections blocks.

It is a DomainError if blocklist includes no citecollections blocks.
